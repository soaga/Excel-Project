# Bike Sales Analysis (Excel)

## Project Overview
This project analyzes bike sales data to understand customer purchasing behavior, demographics, and key factors influencing sales performance.

## Tools Used
- Microsoft Excel
- PivotTables
- Charts & Graphs
- Data Cleaning Functions

## Objectives
- Identify trends in bike purchases based on customer demographics
- Analyze income, age, and commute distance patterns
- Visualize sales performance using dashboards

## Key Insights
- Higher income customers are more likely to purchase bikes
- Middle-aged customers show the highest purchase rate
- Commute distance influences buying decisions

## Project Files
- Bike Sales Dataset
- Excel Dashboard
